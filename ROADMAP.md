# 🗺️ 매매일지 개발 현황 & 로드맵

주식 투자를 **'감'에서 '시스템'으로** 바꾸기 위한 매매일지. 평균단가·다중통화·실제 계좌가치까지
추적해 "진짜 통계"를 내는 것을 목표로 한다. (사용자/배포 안내는 [README.md](README.md) 참고)

> 측정할 수 없으면 관리할 수 없고, 관리할 수 없으면 개선할 수 없다.

---

## ✅ 완료된 기능

### 기반 · 배포
- **Streamlit Community Cloud** 무료 공개 URL — 친구는 링크만 클릭.
- 데이터는 **각자 본인 구글시트**에 저장(gspread 서비스 계정). 비공개·영속성은 본인 구글 계정 책임.
- **개선 워크플로**: 전용 venv · `smoke_test.py`(39개 자동 점검) · `run_local.ps1`/`check.ps1` · 친구용 README.
- 비밀(서비스계정 키·로그)은 `.gitignore` 처리(커밋 안 됨).

### 기록 · 입력
- **3개 탭**: 📊 대시보드 / ✍️ 매매 기록 / 📜 히스토리. `st.fragment`+`st.form`으로 입력 중 깜빡임 제거.
- **빠른 입력 폼** — 한 건씩 입력→즉시 저장(거래 많아도 스크롤 불필요). 성공 후 티커·가격·수량·태그·메모만 비움.
- **전체 거래 수정·삭제 표**(접이식 `data_editor`) — 벌크 편집·삭제·CSV 확정용.
- **종목 검색** — 한국/미국 선택, 개별주식·ETF, 이름·6자리코드·심볼 → 야후 티커 자동변환(FinanceDataReader).
- **CSV 일괄 가져오기**(양식 다운로드→업로드).
- **매매 사유 태그**(계획/추격/충동/패닉/분할/리밸런싱/기타) **+ 메모**.
- **예수금 보정 도구** — 입출금 기록이 없을 때 현재 실제 예수금에 맞춰 보정 입출금 1건 자동 추가.

### 계산 엔진 (`journal_core` · 순수 로직)
- **평균단가(평단)** 방식 · 실현/미실현 손익 · 누적손익.
- **하이브리드 평가** — 체결가는 수동 입력, 보유분은 일별 종가(단타=당일청산이면 종가평가 자동 비활성).
- **같은 시각 BUY/SELL은 매수 먼저 처리**(초과매도 오인 방지).
- **다중통화 USD/KRW 분리** — 통화별 독립 계좌(섞어 합산 안 함).
- **입출금(DEPOSIT/WITHDRAW) → 현금잔고 + 보유시가 = 실제 계좌가치**.

### 시각화 · 분석
- **계좌가치 추이 곡선**(기간 필터) · **자산배분 도넛**(종목 시가 + 현금).
- **종목별 손익표** — 보유수량/평단/보유원가/누적매수금액/실현·미실현/수익률.
- **종목별 누적손익 곡선 + 매수(▲)/매도(▼) 시점 마커**(호버 시 체결가·수량·누적손익).
- **🧠 태그별 매매 피드백** — 태그별 빈도·실현손익·평균·승률 + 막대그래프(실현손익은 매도 시점 태그 기준).

## 🧭 추후 개발 방향 (투자 4관점 기준)

매매일지를 쓰는 4가지 이유에 맞춰 우선순위를 잡는다.

| # | 관점 | 상태 | 다음 작업 |
|---|---|---|---|
| 1 | **진짜 수익률**(외부자금 제외) | ⬜ | **TWR(시간가중수익률)** 곡선 + 벤치마크(S&P500/KOSPI) 비교·알파 ← **0순위 추천** |
| 2 | **심리 제어**(뇌동매매 방지) | ✅ 1차 | 과매매(빈도) 경고, 태그별 손익 **추이**, 메모 검색 |
| 3 | **승률·손익 통계** | ⬜ | 승률·손익비·기대값·**MDD**, **단타(당일청산) vs 장기 분리** 성과 |
| 4 | **포지션 사이징·자산배분** | 🟡 통화별만 | **통화 통합(환율 USDKRW)**, 종목 **집중도 경고**, 현금비중 추이 |

**왜 TWR이 0순위인가**: 지금 수익률 = 총손익 ÷ 누적매수금액(단순 ROI)이라, 입금을 더 하면 자산은 늘지만
"내 운용 실력"과 "그냥 돈을 더 넣은 것"이 섞인다. TWR은 입출금 타이밍을 중화해 **순수 운용 실력**을 잰다.
계좌가치 곡선·입출금 타임라인이 이미 있어 엔진 확장만으로 구현 가능.

## 🗂 개발 이력 (마일스톤)

1. **초기** — Streamlit + 각자 구글시트 매매일지(평단·실현손익).
2. **워크플로** — smoke_test 게이트 + 편의 스크립트 + 친구용 README.
3. **입출금·계좌가치·자산배분** + CSV 가져오기 + 기간 필터.
4. **다중통화(USD/KRW)** 분리.
5. **종목 검색**(국내 이름/코드 → 미국 → ETF → 한/미 분리, FinanceDataReader).
6. **3개 탭 + `st.fragment`/`st.form`** 리팩토링(깜빡임 제거).
7. **fix**: 같은 시각 매수-먼저 정렬(초과매도 오인).
8. **종목별 누적손익 매수/매도 마커**.
9. **리팩토링** — 스키마 단일화 + `render_currency_section` 분할 + 헬퍼 테스트.
10. **빠른 입력 폼**(스크롤 없이 한 건씩).
11. **매매 사유 태그·메모 + 태그별 심리 피드백**(관점 2).
12. **예수금 보정 도구**(입출금 기록 없을 때).
